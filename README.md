# Digital Clock with Theme Selector

This is a **digital clock project** built with **HTML, CSS, and JavaScript**.  
It shows the current time and date, and allows users to **change the theme** of the clock. The selected theme is saved in the browser so it stays even after refreshing the page.

---

## Features

- Real-time clock with **hours, minutes, and seconds**
- Displays **day, date, and month**
- **5 themes**: Black, Purple, Blue, Orange, Green
- **Theme saved** using `localStorage` (remains after refresh)
- Fully **responsive design** for desktop, tablet, and mobile
- Clickable **color circles** for easy theme switching

---

## How to Use

1. Open `index.html` in any browser
2. The clock starts automatically
3. Click on the colored circles to change the theme
4. The theme will stay even after refreshing the page

---

## Built With

- **HTML** – structure  
- **CSS** – styling & responsiveness  
- **JavaScript** – clock logic & theme handling
